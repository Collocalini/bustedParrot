Я сделал конвертер из fodg в htic. Это форматы такие а не кот прошёл по клавиатуре.
I made a converter from fodg to htic. It wasn't a cat walking over the keyboard, those are file formats.
htic.js это библиотека для создания интерактивных страниц на подобии twinery.org. А hticExt позволяет нарисовать схему, описывающую интерактивные переходы, в LibreOffice Draw и затем сконвертировать её в интерактивный html.
htic.js is a library for creating interactive pages similar to twinery.org. And hticExt allows to draw a scheme describing interactive transitions the page should have using LibreOffice Draw and then build the interactive html.

hticExt
Сборка конвертера под Debian x86. Но должно работать на любом современном линуксе(кроме nixos). На самом деле понятия не имею с чем эта сборка совместима, собирайте из исходников.
The build for Debian x86. But it should work on any modern linux(but not on nixos). I actually have no idea what it will be compatible with. Build from source if necessary.
